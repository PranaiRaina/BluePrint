# BoostUp - Agentic Financial Dashboard

The "Midnight Emerald" frontend for your RAG Pipeline.

## Setup
1. Open a new terminal.
2. Navigate to this folder:
   ```bash
   cd frontend
   ```
3. Install dependencies (if you haven't):
   ```bash
   npm install
   ```
4. Start the development server:
   ```bash
   npm run dev
   ```
5. Open `http://localhost:5173` to see the Agent in action!

## Features
- **Project BoostUp Design**: Deep Navy, Emerald, Violet.
- **Glassmorphism UI**: Beautiful transparent cards.
- **"Lifting" Search**: Type a query and watch the interface transform.
- **Dynamic Zones**: Hard Data, Agent Insights, and Projections.
